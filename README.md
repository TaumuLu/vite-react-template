# vite-react-template

## 目录结构

- docs：开发文档记录
- env：统一存放环境变量目录
- scripts：脚本
- src
  - assets：资源文件
  - components：公共组件
  - constant：常量、枚举、默认值等
  - router：路由
  - styles：全局公共样式
  - types：ts 类型声明
  - utils：工具函数

### 移动端开发

- 移动端路由文件以 .mobile.tsx 结尾，动态加载会判断当前端加载对应的 index 文件

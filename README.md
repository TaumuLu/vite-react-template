# vite-react-template

## 目录结构

- env：统一存放环境变量目录
- scripts：脚本
- src
  - assets：资源文件
  - components：公共组件
  - pages：路由页面
  - constant：常量、枚举、默认值等
  - router：路由配置
  - styles：全局公共样式
  - types：ts 类型声明
  - utils：工具函数
